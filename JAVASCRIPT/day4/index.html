Lesson 4: Variables

Main Concepts

Variables store data values. In modern JavaScript we use let and const. The old keyword var is rarely used today.

  • let: allows reassignment (mutable).
  • const: cannot be reassigned (immutable).
  • var: outdated, avoid in modern code.

Rules for Naming

  • Names must contain only letters, digits, $ and _.
  • First character cannot be a digit.
  • Use camelCase for multi-word names (example: myVariable).
  • Case sensitive (age and Age are different).
  • Avoid reserved words like let, class, etc.

Constants

Use const when the value should never change. For real constants like colors or fixed values, write them in ALL_CAPS.
