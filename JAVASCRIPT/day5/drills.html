JavaScript Data Types - Day 5 Drills

Drill 1 – Numbers

  1. Create variables price1 = 15.567 and price2 = 23.456.
  2. Multiply them and show the total rounded to 1 decimal place.
  3. Divide price2 by price1 and round to 2 decimal places.
  4. Use alert() for both results.

Drill 2 – BigInt

  1. Declare bigA = 9007199254740991000n and bigB = 12345678901234567890n.
  2. Add them and subtract them, showing both results using alert().

Drill 3 – Strings / Template Literals

  1. Declare firstName = "Ameer", lastName = "Hamza", skill = "Gaming".
  2. Using template literals, show:
    My name is Ameer Hamza. I love Gaming.
  3. Concatenate firstName + lastName manually and compare with template literal output.

Drill 4 – Boolean

  1. Declare age = 25, minAge = 18, maxAge = 30.
  2. Create a boolean variable isAllowed that is true if age is between minAge and maxAge.
  3. Alert isAllowed.

Drill 5 – Null / Undefined

  1. Declare let x; and let y = null;
  2. Alert both and check their types using typeof.
  3. Bonus: Add a comment explaining why typeof null returns "object".

Drill 6 – Mixed Types

  1. Declare let a = 10 and let b = "10".
  2. Compare a == b and a === b.
  3. Alert both results and explain the difference in a comment.
