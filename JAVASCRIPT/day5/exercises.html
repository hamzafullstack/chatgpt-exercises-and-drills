JavaScript Data Types - Day 5 Challenges

1. Number Basics

Multiply price and quantity and make sure total shows 2 decimal places:

let price = 19.99;
let quantity = 3;
let total = price * quantity;
alert("Total cost: " + total);

2. BigInt

Fix the code so it actually handles this as a BigInt:

let big = 123456789012345678901234567890;
alert(big);

3. String Interpolation

Make a sentence using backticks and ${} instead of +:

let user = "Hamza";
let age = 25;
let hobby = "Gaming";
alert("Hello I am " + user + ", my age is " + age + " and I love " + hobby);

4. Boolean Comparisons

Declare a variable isAdult which is true if age >= 18:

let age = 25;
let isAdult;
alert(isAdult);

5. Null and Undefined

Declare variables for null and undefined and check them:

let userName;
let car = null;
alert(userName);
alert(car);

6. typeof Checks

Predict the type of different values using typeof:

alert(typeof null);
alert(typeof undefined);
alert(typeof isAdult);
alert(typeof big);
alert(typeof hobby);
