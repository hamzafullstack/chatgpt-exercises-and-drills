Exercise Two by chatgpt

Your Task / Homework (Drill-Style)

Task: Create a small JavaScript snippet that:

  1. Displays 3 alert messages—one by one.
  2. Includes comments that explain each step.
  3. Intentionally uses semicolons (avoid implicit rules).
  4. Then, introduce one semicolon omission or misuse that breaks the code, and comment out that broken line.

You must submit:

  • A .js code block containing:
  • functional code with proper semicolons and comments.
  • A commented-out broken variant clearly labeled as "BROKEN—demonstrates bad semicolon usage", plus a comment explaining why it fails.

This tests your understanding of:

  • Statements
  • Proper semicolon usage vs. automatic insertion edge-case
  • Comments usage

Follow These Steps:

  1. Write the working version first
  2. Add one broken version, commented out
  3. Submit the code snippet exactly as above in your message.
