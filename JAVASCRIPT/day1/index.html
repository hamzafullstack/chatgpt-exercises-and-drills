  • Create a new HTML file.

    • Inline script that shows an alert "Inline Hello"..
    • External script file test.js that shows an alert "External Hello".

    Open the page in your browser and observe:

    • What happens if you swap the order of the script tags?.

    inline hello alert popups first then external

    if you swap the order of the script tags, external alert pops up first then inline

    this is because the inline script is executed immediately, while the external script is loaded and executed after the inline script.

    the external script is loaded asynchronously, so it does not block the execution of the inline script.

    this is a fundamental concept in JavaScript, and it is important to understand how scripts are executed in the browser.
